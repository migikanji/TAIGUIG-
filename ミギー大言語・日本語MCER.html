は、が、の、も、をの助詞

は (wa)

La partícula **は** se pronuncia "wa" cuando se utiliza en las oraciones, aunque se escribe como "は" en hiragana. **は** se usa para marcar el **tema** de la oración. En otras palabras, **は** introduce lo que se va a hablar, mientras que lo que sigue en la oración es el **enfoque principal**. Piensa en ella como una forma de señalar de qué estás hablando.La estructura es algo como: tema は enfoque principal. Aquí tienes un ejemplo:

私（わたし）はミゲルです。 Watashi wa Miguel desu. = Yo soy Miguel. En este caso, "yo" es el tema de la oración, y "soy Miguel" es la información principal sobre el tema. La partícula **は** señala que el foco de la oración es "yo".

が

La partícula **が** se usa para marcar el **enfoque principal** de la oración, es decir, lo que **es importante** o lo que se quiere enfatizar. A menudo, el **sujeto** de la acción también está marcado por **が**, pero lo importante es que marca el **enfoque** de la acción o descripción, en lugar de solo identificar el sujeto. Un ejemplo es:
私がミゲルです ＝ *YO soy miguel* En esta oración, el enfoque principal es *yo* y no *miguel*, por lo que usamos **が** para indicar que *yo* es el que tiene la importancia
Otro ejemplo: 犬が好きです ＝ *Me gustan los perros*. Aquí, el enfoque principal es *los perros* (犬), ya que es lo que se enfatiza sobre lo que me gusta.

の

La partícula の se usa para indicar **posesión** o para hacer una **relación** entre dos sustantivos. Se traduce como "de" o "de pertenencia". Un ejemplo es:
私の本 ＝ *Mi libro*. En este caso, *私* (yo) está relacionado con *本* (libro) a través de la partícula の, lo que indica que el libro me pertenece.

も

La partícula も se usa para indicar "también" o "inclusive". Se usa para añadir algo similar o relacionado con lo que se ha mencionado antes. Un ejemplo es:
私は学生です。彼も学生です ＝ *Yo soy estudiante. Él también es estudiante*. Aquí, la partícula も indica que la segunda persona también es estudiante, al igual que el sujeto anterior.

を (o)

La partícula を se usa para indicar el **objeto directo** de una acción, es decir, lo que está siendo hecho o afectado por el verbo. Un ejemplo es:
本を読む ＝ *Leer un libro*. Aquí, *本* (libro) es el objeto directo de la acción *leer* (読む). La partícula を se coloca después del objeto directo.
へ、に、で、から、まで、か、と、よ、やの助詞

へ (e)

La partícula へ indica dirección o destino, y se traduce como **"a"** o **"hacia"**.

Ejemplo: 学校へ行く (がっこうへいく) = *Voy a la escuela*.

En este caso, "学校" (escuela) es el destino al que se va.

で (de)

La partícula で se usa para indicar el lugar donde ocurre una acción o el medio de transporte.tambien para decir con que haces la accion

Ejemplo: 公園で遊ぶ (こうえんであそぶ) = *Jugar en el parque*.

También se puede usar para medios de transporte: バスで行く (ばすでいく) = *Voy en autobús*.

con que haces la accion: 鉛筆で書く (えんぴつでかく) = *escribo con el lapiz*.

から (kara) / まで (made)

から se usa para indicar el **punto de inicio** de una acción o evento, y まで se usa para indicar el **punto final**.

Ejemplo: 9時から仕事を始めます (くじからしごとをはじめます) = *Empiezo a trabajar desde las 9*.

Ejemplo: 5時まで働きます (ごじまではたらきます) = *Trabajo hasta las 5*.

か (ka)

La partícula か se utiliza para hacer preguntas o para dar opciones entre dos o más cosas. Se traduce como **"?"** en preguntas o **"o"** para opciones.

Ejemplo: 日本へ行きますか (にほんへいきますか) = *¿Vas a Japón?*.

Ejemplo: コーヒーかお茶を飲みますか (こーひーかおちゃをのみますか) = *¿Vas a tomar café o té?*.

よ (yo)

La partícula よ se coloca al final de la oración para dar énfasis a lo que se dice. Puede expresar que la información es nueva para el oyente o agregar certeza a lo que se afirma.

Ejemplo: これは面白いよ (これはおもしろいよ) = *Esto es interesante, ¡te lo aseguro!*.

En este caso, よ le da énfasis a la afirmación de que algo es interesante, como una forma de asegurarle al oyente que lo es.

と (to)

La partícula と se usa para conectar sustantivos, y se traduce como **"y"** o **"con"**. Es muy útil para hacer listas o para indicar compañía.

Ejemplo: 猫と犬 (ねこといぬ) = *El gato y el perro*.

Ejemplo: 友達と行く (ともだちといく) = *Voy con un amigo*.

や (ya)

La partícula や se usa para enumerar cosas de manera no exhaustiva, similar a "y" en español, pero indicando que hay más cosas no mencionadas.

Ejemplo: リンゴやバナナ (りんごやばなな) = *Manzanas y plátanos (entre otras cosas)*.

へ (e) vs. に (ni)

Ambas partículas se usan para indicar dirección o destino, pero へ se usa para expresar el sentido de "hacia", mientras que に se usa cuando se especifica el destino exacto o una acción en ese lugar.

Ejemplo: 学校に行く (がっこうにいく) = *Ir a la escuela* (especificando la acción de llegar).

Ejemplo: 公園へ行く (こうえんへいく) = *Ir al parque* (dirección, sin especificar la acción de llegar).
